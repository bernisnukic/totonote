# Keyboard shortcuts

Shortcuts are written below as **macOS** / **Windows & Linux**.

## Text formatting

| Action | macOS | Windows & Linux |
|--------|-------|-----------------|
| Bold | `⌘B` | `Ctrl+B` |
| Italic | `⌘I` | `Ctrl+I` |
| Underline | `⌘U` | `Ctrl+U` |
| Strikethrough | `⌘⇧S` | `Ctrl+Shift+S` |
| Inline code | `⌘E` | `Ctrl+E` |

## Headings and blocks

| Action | macOS | Windows & Linux |
|--------|-------|-----------------|
| Heading 1 | `⌘⌥1` | `Ctrl+Alt+1` |
| Heading 2 | `⌘⌥2` | `Ctrl+Alt+2` |
| Heading 3 | `⌘⌥3` | `Ctrl+Alt+3` |
| Back to normal text | `⌘⌥0` | `Ctrl+Alt+0` |
| Bullet list | `⌘⇧8` | `Ctrl+Shift+8` |
| Numbered list | `⌘⇧7` | `Ctrl+Shift+7` |
| Quote | `⌘⇧B` | `Ctrl+Shift+B` |
| Code block | `⌘⌥C` | `Ctrl+Alt+C` |

Bold, italic, underline, strikethrough, the three headings and both list types also have
buttons in the toolbar at the top of the window. Quote, code block and inline code are
keyboard-only.

## Working with lists

| Action | Key |
|--------|-----|
| New item | `Enter` |
| Indent (make it a sub-item) | `Tab` |
| Outdent | `Shift+Tab` |

## Line breaks

`Shift+Enter` starts a new line inside the same paragraph, instead of starting a new
paragraph.

## Undo and redo

| Action | macOS | Windows & Linux |
|--------|-------|-----------------|
| Undo | `⌘Z` | `Ctrl+Z` |
| Redo | `⌘⇧Z` or `⌘Y` | `Ctrl+Shift+Z` or `Ctrl+Y` |

## Elsewhere in the app

| Key | What it does |
|-----|--------------|
| `Escape` | Closes the open dialog, or a right-click menu |
| `↑` / `↓` | Moves through tags in the left sidebar, once you've clicked one |

## Saving

There is no save shortcut, because there is nothing to save manually — your writing is
saved automatically about a second after you stop typing. The status bar in the bottom
right tells you when it has.

## About the shortcut list in Settings

Settings has a **Keyboard Shortcuts** section that lists shortcuts and appears to let you
rebind them. That list is **informational only at the moment** — changing a binding there
is remembered, but the editor still responds to the keys in the tables above. Two of its
entries are also out of date: it shows Strikethrough as `⌘⇧X` (the working key is `⌘⇧S`)
and a Save shortcut that does not exist.

The tables on this page describe what actually works today.
